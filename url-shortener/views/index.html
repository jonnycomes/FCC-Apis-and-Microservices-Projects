URL Shortener Microservice

How this site works

  • You can POST a URL to https://quiver-icon.glitch.me/api/shorturl/new using the input field below, and you will receive a shortened URL in the JSON response.
  • If you pass an invalid URL that doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}
  • When you visit the shortened URL, i.e. https://quiver-icon.glitch.me/api/shorturl/[shortened_url], it will redirect you to the original link.
