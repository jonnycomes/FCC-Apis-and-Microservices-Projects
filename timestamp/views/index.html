Timestamp Microservice

This site provides a timestamp API using the following endpoint:

GET https://savory-pyrite.glitch.me/api/timestamp/:date_string?
  • For valid date strings, the api returns a json containing the corresponding unix and utc timestamps.
  • If the date string is empty, the api will return the timestamps for the current time.
  • For invalid date strings, the api returns the following json:
    {"unix": null, "utc" : "Invalid Date"}

Example Usage

Click on the following examples to see the api in action:

  • GET .../api/timestamp/2015-12-25
  • GET .../api/timestamp/Jan 14 2014
  • GET .../api/timestamp/1450137600
  • GET .../api/timestamp/2007-06-09T17:46:21
  • GET .../api/timestamp
